# AA Annotate v0.1.1

Refinement release for the Windows annotation overlay, agent handoff artifacts, and bundled skill.

## Changed

- Added annotated overview images with numbered boxes for each exported capture.
- Added cropped image snippets for each exported annotation.
- Treat cropped captures as the primary exported image and export coordinates relative to that image.
- Drop annotations fully outside the crop on export, clip partially crossing annotations, and renumber exported annotations.
- Added visual indicators for annotations that will be dropped by crop export.
- Improved comment editing with Enter-to-save, Esc-to-cancel/delete behavior, and a more compact panel.
- Kept captured screenshots hidden while only idle chrome, capture lists, or About are open.
- Refined the compact command bar, capture list, display selector, About panel, and annotation box styling.
- Added minimum annotation-size feedback while drawing small boxes.
- Updated the bundled Codex skill to explain exported images, snippets, crop behavior, and coordinate rules.

## Fixed

- Prevented screenshot background leakage behind idle dropdown and About panels.
- Fixed unreadable comment text selection styling.
- Fixed small annotation boxes being exported unintentionally.
- Fixed annotation selection behavior for overlapping boxes.

## Install

One-command install:

```powershell
powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/adiladiloglu/AA.Annotate/master/install.ps1 | iex"
```

Manual install:

Download `aa-annotate-0.1.1-win-x64.zip`, extract it, and run:

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1
```
