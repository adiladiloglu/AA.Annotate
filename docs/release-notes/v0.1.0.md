# AA Annotate v0.1.0

Initial Windows release of AA Annotate.

## Included

- Avalonia desktop annotation overlay for Windows.
- Agent-facing CLI: `aa-annotate session --wait`.
- Multi-capture sessions.
- Display selection and idle command-bar movement.
- Screenshot capture, crop, blur mask, numbered annotation boxes, and comments.
- OS temp directory session storage.
- Generated `review.md` handoff for agents.
- Structured `annotations.json` for exact coordinates.
- Codex skill package.
- User-local PowerShell installer and uninstaller.

## Install

Download `aa-annotate-0.1.0-win-x64.zip`, extract it, and run:

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1
```
